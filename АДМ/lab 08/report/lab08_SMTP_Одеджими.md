# Контрольные вопросы

1. В каком каталоге и в каком файле следует смотреть конфигурацию Postfix?  
   Основная конфигурация Postfix располагается в каталоге /etc/postfix.  
   Главный конфигурационный файл — /etc/postfix/main.cf, дополнительные параметры служб задаются в /etc/postfix/master.cf.

2. Каким образом можно проверить корректность синтаксиса в конфигурационном файле Postfix?  
   Для проверки корректности конфигурации используется команда postfix check.  
   Дополнительно корректность применённых параметров можно контролировать через postconf, который выводит значения параметров в интерпретированном виде, что позволяет выявлять некорректные записи.

3. В каких параметрах конфигурации Postfix требуется внести изменения в значениях для настройки возможности отправки писем не на локальный хост, а на доменные адреса?  
   Для обеспечения отправки/доставки писем на доменные адреса требуется настройка следующих параметров:  
   - inet_interfaces — разрешает Postfix принимать соединения не только с localhost, но и с других интерфейсов (например, inet_interfaces = all).  
   - mynetworks — задаёт диапазоны доверенных сетей, которым разрешено пересылать почту через сервер (например, 127.0.0.0/8 и внутренняя сеть 192.168.0.0/16).  
   - mydestination — определяет домены, для которых данный сервер является конечной точкой доставки (включение собственного домена позволяет доставлять письма на user@user.net локально).  
   Также для корректной доменной доставки требуется наличие корректных DNS-записей (A/MX и PTR) для домена и почтового хоста.

4. Приведите примеры работы с утилитой mail по отправке письма, просмотру имеющихся писем, удалению письма.  
   - Отправка письма:  
     echo . | mail -s test1 akabrikosov@server.akabrikosov.net  
   - Просмотр списка писем в почтовом ящике (в интерактивном режиме):  
     mail  
     После запуска будут показаны письма с номерами, далее можно открыть письмо командой с номером, например: 1  
   - Удаление письма в интерактивном режиме mail:  
     d 1  
     (удаляет письмо с номером 1)  
     Затем выход с сохранением изменений:  
     q  
   - Просмотр почтового ящика напрямую через spool-файл:  
     cat /var/spool/mail/akabrikosov  

5. Приведите примеры работы с утилитой postqueue. Как посмотреть очередь сообщений? Как определить число сообщений в очереди? Как отправить все сообщения, находящиеся в очереди? Как удалить письмо из очереди?  
   - Просмотр очереди сообщений:  
     postqueue -p  
   - Определение количества сообщений в очереди:  
     postqueue -p | grep -E '^[A-F0-9]' | wc -l  
     (подсчёт идентификаторов сообщений в выводе очереди)  
   - Принудительная отправка всех сообщений из очереди:  
     postqueue -f  
   - Удаление конкретного письма из очереди по ID:  
     postsuper -d <ID_сообщения>  
   - Очистка всей очереди:  
     postsuper -d ALL
